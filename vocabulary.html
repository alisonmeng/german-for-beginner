Everyday Vocabulary

These words that come with pronunciation guide will take you places.

Jump to Greetings, Weather, People, or Home

Greetings

  • Hello / Hallo / hâ-loh
  • Goodbye / Auf Wiedersehen / ouf vee-der-zey-en
  • Bye / Tschüs / chues
  • Pardon? / Wie bitte? / vee bit-e?

Weather

  • sunny / sonnig / son-iH
  • hot / heiß / hays
  • windy / windig / vin-diH
  • ready / Fertig / fêrt-iH

People

  • mother / die Mutter / dee moot-er
  • father / der Vater / dêr fah-ter
  • wife / die Frau / dee frou
  • husband / der Mann / dêr mân
  • kids / die Kinder / dee kin-der

Home

  • living room / das Wohnzimmer / dâs vohn-tsi-mer
  • kitchen / die Küche / dee kueH-e
  • bathroom / das Badezimmer / dâs bah- de-tsi-mer
  • bedroom / das Schlafzimmer / dâs shlahf-tsi-mer
